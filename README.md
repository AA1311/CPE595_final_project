# Welcome to the Car Price Estimator tool!
---
This web tool uses AI algorithms to help you get an instant price estimate on any vehicle!
**How to use:**
1. Download the requirements.txt file, use "python -m pip install -r requirements.txt" in cmd to install the file.
2. In the same folder, download app.py, car dataset cleaned.xlsx, car_price_prediction_improved.py, and style.css in the same folder.
3. Type `streamlit run app.py` in the terminal.
4. A window for the app should automatically open in your web browser, if it does not follow the instructions in the terminal to view the app in your browser.
5. It will take a minute for the app to load while the algorithms train.
6. Now enter your car information into the app and enjoy!

This project uses machine learning models (Linear Regression, Decision Tree, and Random Forest) to predict used car prices based on vehicle features. These models were chosen to capture both linear and nonlinear relationships within the data, with Random Forest providing the most accurate predictions.
